## Coding challenge - ID

### Run locally

Clone the project

```bash
  git@github.com:emmolka/ignite.git
```

Go to the project directory

```bash
  cd /ignite
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

### Run tests

```bash
  npm run test
```

### Run using docker

```bash
  docker build -t react-image .
```

```bash
  docker run --rm -p 3000:3000 react-image
```
